# Sales Data Cleaning & Visualization Project

## Tools Used
- Python (Pandas)
- Power BI

## Project Overview
This project focuses on cleaning raw sales data using Python and visualizing insights using Power BI.

## Data Cleaning (Python)
- Removed missing values
- Checked dataset structure
- Handled null values
- Exported cleaned dataset

## Visualization (Power BI)
- Total Sales, Orders, Customers
- Sales by Category
- Sales by Region
- Top 5 Products
- Sales Distribution (Boxplot)

## Files Included
- project.py → Data cleaning code
- cleaned_data.csv → Cleaned dataset
- Sales-Dashboard.pbix → Power BI dashboard
- overview.png → Dashboard overview
- analysis.png → Analysis page

## Outcome
Cleaned dataset and created an interactive dashboard to derive business insights.
